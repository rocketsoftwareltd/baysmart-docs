Edit Project

To edit a project, search for it in the project datat table by using the search box or navigating through the table's pagination; next click the edit icon under the "actions" column and the edit form will be displayed on a modal and a form prefilled with existing data about the project for you to edit. Click the blue button to save when you're done.

Bulk upload project

Rather than make entries for the projects one by one, a bulk of it can be uploaded via a csv file. To upload it, click the blue "bulk upload" button to display a modal as shown below. Select the csv file and click upload.

View project sheet

The project sheet shows you the information and details associated a single project, to view it, click the "eye" icon of the project you want to view under the action column of the projects list datatable. The page will show you the project's data available at the time of viewing it.

Delete project

To delete a project, simply click the delete icon under the actions column and click "confirm delete" in the modal to delete.

Beneficiaries

To arrive at the beneficiaries page, under the project Tools menu, click the beneficiaries sub menu and it will bring you to the beneficiaries page as shown below. On the beneficiary page, you'll be able to select a particular project and then select the beneficiary type and enter the number of beneficiaries for that particular project. You can also manage beneficiary types as shown in the screenshot below to the right. You can add to the default types of beneficiaries, which are individuals, households, communities and LGA beneficiaries. To add a new beneficiary, click the Add beneficiary button at the top left corner of the page. A modal will pop up and under model you'll see a form where you can select a project you're trying to add beneficiaries for. Then you click Save, to say the beneficiaries
