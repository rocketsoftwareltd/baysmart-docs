General

Does baysmart have any warranty?

Yes baysmart has a one year warranty.

Why do I see the "Page Expired" message sometimes?

This happens when a page in baysmart has been left open but inactive for some time, the system assumes you've left hence the "Page Expired" message for security reasons.

How do I change my password?

To change your password go to the profile page, enter your new password in the update profile page and click the update button.

How secured is baysmart?

Baysmart very secured from cyber attacks because it's build with security features like CSRF (Cross Site Request Forgery), XSS (Cross Site Scripting) and SQL Injection

Will the technologies behind baysmart be updated?

Yes, whenever the programming language, Database system, Virtual servers and all other technologies involed in building baysmart release a new version, we'll update them immediately.
